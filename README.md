# Clustering COVID-19 Indonesia

## Deskripsi
Proyek ini bertujuan mengelompokkan provinsi di Indonesia berdasarkan dinamika kasus COVID-19 bulanan selama periode 2020–2022 menggunakan pendekatan **unsupervised learning**. Hasil clustering digunakan untuk mengidentifikasi karakteristik dan pola perkembangan pandemi pada setiap provinsi.

## Tujuan
- Melakukan eksplorasi data kasus COVID-19 di Indonesia.
- Melakukan preprocessing data sebelum proses clustering.
- Menentukan jumlah cluster optimal.
- Mengelompokkan provinsi berdasarkan karakteristik perkembangan kasus COVID-19.
- Menginterpretasikan hasil clustering untuk memperoleh insight.

## Dataset
- Data kasus COVID-19 bulanan provinsi di Indonesia (2020–2022).

## Metode
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- K-Means Clustering
- Elbow Method
- Silhouette Score
- Visualisasi Hasil Clustering

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Hasil
Model berhasil mengelompokkan provinsi berdasarkan pola perkembangan kasus COVID-19 sehingga memberikan gambaran mengenai karakteristik masing-masing kelompok wilayah selama periode pandemi.
